This repository contains my work on *HTB COAE Module 2: Applications of AI in InfoSec*

## What is HTB COAE?
[HTB COAE](https://academy.hackthebox.com/preview/certifications/htb-certified-offensive-ai-expert) is a hands-on cybersecurity certification assessing advanced skills in red teaming and exploiting AI and Machine Learning (ML) systems.
